# 🚀 Módulo 04: JavaScript Avanzado y POO

Este repositorio contiene las actividades de aprendizaje, enfocadas en Programación Orientada a Objetos (POO), manipulación del DOM, funciones Callback, Promesas y consumo de APIs REST.

## 🛠️ Tecnologías utilizadas
- **JavaScript (ES6+)**
- **HTML5 & CSS3**
- **Bootstrap 5** (Framework de diseño)
- **Git & GitHub** (Control de versiones)
